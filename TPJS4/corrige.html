Corrigé des Questions

1. Quelle est la couche du modèle OSI responsable du routage des paquets ?

Réponse correcte : Couche 3 - Réseau

2. Quel est le rôle principal d'un pare-feu dans une architecture réseau ?

Réponse correcte : Filtrer le trafic entrant et sortant

3. En programmation, que signifie le principe DRY (Don't Repeat Yourself) ?

Réponse correcte : Éviter de dupliquer le code

4. Dans une base de données relationnelle, qu'est-ce qu'une clé primaire ?

Réponse correcte : Une colonne qui identifie de manière unique une ligne

5. Que signifie l'acronyme HTTPS ?

Réponse correcte : HyperText Transfer Protocol Secure

6. En gestion de projet, que représente un diagramme de Gantt ?

Réponse correcte : Un planning des tâches du projet

7. Quelle commande Git permet de copier un dépôt distant sur votre machine locale ?

Réponse correcte : git clone

8. Dans un réseau local, quel rôle joue un serveur DHCP ?

Réponse correcte : Il attribue des adresses IP dynamiques

9. Quelle est la différence principale entre HTTP et HTTPS ?

Réponse correcte : HTTPS chiffre les données échangées

10. Quel est l'objectif du responsive design en développement web ?

Réponse correcte : Créer des interfaces web responsives
